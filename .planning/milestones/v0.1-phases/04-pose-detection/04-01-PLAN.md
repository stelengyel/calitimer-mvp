  <action>
Create a new `CaliTimer/Vision/` directory and `VisionProcessor.swift`.

`DetectedPose` is a `Sendable` struct holding a `[String: CGPoint]` dictionary keyed by joint name strings (not `VNHumanBodyPoseObservation.JointName` to avoid Sendable issues across actor boundaries). Joints are stored as normalized Vision coordinates (0,0 bottom-left, 1,1 top-right).

`VisionProcessor` is `@CameraActor`-isolated, `ObservableObject`. It holds:
- `private let requestHandler = VNSequenceRequestHandler()` — created once, reused per-frame
- `@Published nonisolated(unsafe) var detectedPose: DetectedPose? = nil` — published from CameraActor, read on MainActor. Use `nonisolated(unsafe)` so SwiftUI bindings can read it without crossing actor. Update via `Task { @MainActor in self.detectedPose = result }` from inside the CameraActor-isolated processing method.

Public method: `func process(sampleBuffer: CMSampleBuffer)` — isolated to `@CameraActor`. Creates `VNDetectHumanBodyPoseRequest`. Performs request via `requestHandler.perform([request], on: sampleBuffer)`. On success, extracts these 8 joints only (ignore all others to reduce clutter and match handstand detection scope):
- `.leftWrist`, `.rightWrist`
- `.leftShoulder`, `.rightShoulder`
- `.leftHip`, `.rightHip`
- `.leftAnkle`, `.rightAnkle`

For each joint, check `confidence > 0.2` before including. Store as `[String: CGPoint]` using `joint.rawValue` as key and `VNRecognizedPoint.location` (already normalized) as value. If no observations or zero qualifying joints, publish `nil`.

`DetectedPose`:
```swift
struct DetectedPose: Sendable {
    // Key: VNHumanBodyPoseObservation.JointName.rawValue (String)
    // Value: normalized Vision CGPoint (0,0 = bottom-left)
    let joints: [String: CGPoint]
}
```

Log a single `print("[Vision] detected \(joints.count) joints")` per frame for debug verification. This will be the evidence of DETE-05 correctness during verify.

Add `CaliTimer/Vision/` group to `project.yml` under the `CaliTimer` target sources. Pattern: match existing `Camera`, `Upload`, `UI` entries.

**SkeletonPreference.swift**

`@MainActor final class SkeletonPreference: ObservableObject` with:
- `@Published var isEnabled: Bool` — backed by `UserDefaults.standard.bool(forKey: "skeletonOverlayEnabled")`
- `init()`: reads UserDefaults, defaults to `true` (first launch shows skeleton per context decision)
- `didSet` on `isEnabled`: writes `UserDefaults.standard.set(isEnabled, forKey: "skeletonOverlayEnabled")`

Since `@Published` property observer doesn't trigger on direct set, use `Combine` `.sink` in `init()` to observe `$isEnabled` and write to UserDefaults: `$isEnabled.sink { UserDefaults.standard.set($0, forKey: "skeletonOverlayEnabled") }`. Store cancellable in `private var cancellable: AnyCancellable?`.

**SkeletonOverlayView.swift**

`struct SkeletonOverlayView: View` taking:
- `joints: [String: CGPoint]` — normalized Vision coords
- `viewSize: CGSize` — the pixel size of the view being overlaid (needed for coordinate flip)

Vision normalized coords have (0,0) at BOTTOM-LEFT, y increasing upward. SwiftUI Canvas has (0,0) at TOP-LEFT, y increasing downward. Flip y: `canvasY = (1.0 - normalizedY) * viewSize.height`, `canvasX = normalizedX * viewSize.width`.

Uses SwiftUI `Canvas` as body. No animations. Only renders if `joints` dict is non-empty.

Bone connections to draw (8 connections):
```
("left_wrist_2d", "left_shoulder_2d"),
("right_wrist_2d", "right_shoulder_2d"),
("left_shoulder_2d", "right_shoulder_2d"),   // shoulder crossbar
("left_shoulder_2d", "left_hip_2d"),
("right_shoulder_2d", "right_hip_2d"),
("left_hip_2d", "right_hip_2d"),             // hip crossbar
("left_hip_2d", "left_ankle_2d"),
("right_hip_2d", "right_ankle_2d")
```

Use `VNHumanBodyPoseObservation.JointName` raw values as keys. Draw order: bones first, then dots on top.

Bone style: `Color.brandEmber` stroke, lineWidth 2, opacity 0.85
Joint dot style: `Color.brandEmber` fill, circle radius 4

`Canvas { context, size in ... }` — convert each joint's normalized point to canvas point inside the closure. Skip bones where either endpoint joint is missing from the dict (partial skeleton renders gracefully).

Both files go in `CaliTimer/Vision/` (already added to project.yml in Task 1).
